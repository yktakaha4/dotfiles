## Usage

```
git clone --recursive git@github.com:yktakaha4/dotfiles.git "$HOME/.dotfiles"

ln -s "$HOME/.dotfiles/.zprofile" "$HOME/.zprofile"
ln -s "$HOME/.dotfiles/.zshrc" "$HOME/.zshrc"

zsh
```
